# exports/

One `slop-list/1` JSON file per contributor, submitted by pull request.

Generate yours with the extension: History tab → **Export my list (JSON)**.
Name it after your reporter UUID or a stable pseudonym, e.g.
`9f6c1c1e.json`. Keep re-submitting the same filename as your list grows —
one file per reporter, not per export.

PR requirements (see [docs/LIST_FORMAT.md](../docs/LIST_FORMAT.md)):

- Valid `slop-list/1` format (CI checks this).
- Entries carry evidence links to the actual posts.
- Only accounts you personally reviewed — export from queue mode, not from
  an unattended auto-mode session.
